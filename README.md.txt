# CarSale

 es un sitio web responsivo de compra y venta de autos usados por marca.  
Este proyecto fue realizado como entrega final para la cursada de desarrollo web, poniendo en práctica HTML, Bootstrap, SASS y diseño responsive.

---

 Características principales

- Catálogo de autos divididos por marca (Chevrolet y Citroen)
- Página de inicio con botón de acción destacado
- Página “Sobre Nosotros” con estructura clara y animaciones suaves
- Página de contacto con formulario funcional y datos de empresa
- Estilos aplicados con SCSS utilizando variables, nesting y animaciones
- Diseño responsive para mobile, tablet y desktop

---

 Tecnologías utilizadas

- HTML5 semántico
- Bootstrap 5 (grids, clases utilitarias)
- SASS/SCSS con nesting, variables, funciones y animaciones
- AOS (Animate On Scroll) para efectos suaves
- FontAwesome para íconos
